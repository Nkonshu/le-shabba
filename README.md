# Le Shabba

Plateforme éducative pour l'Afrique francophone (Côte d'Ivoire, Bénin, Togo, Sénégal, Cameroun,
France) — Next.js 16 (App Router, Turbopack), Supabase self-hosted, Meilisearch.

## Démarrage rapide

```
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Documentation

- [`docs/environnement-et-deploiement.md`](docs/environnement-et-deploiement.md) — comment relancer
  l'environnement de dev après un redémarrage, comment mettre la version en ligne à jour, et le
  rôle de Coolify / Tailscale / Cloudflare dans l'infrastructure.
- `AGENTS.md` — instructions pour les agents IA travaillant sur ce dépôt (Next.js 16 a des
  changements de rupture par rapport aux versions précédentes, à lire avant de coder).
